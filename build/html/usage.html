Requirements¶

RAFIKI is a python-based code requiring the following packages: numpy, scipy, cython, h5py, matplotlib, yt, and CAESAR

RAFIKI uses CAESAR outputs and builds on the yt simulation analysis toolkit. We recommend installing yt via Anaconda:

$ conda install -c conda-forge yt

More information on yt and other installation options can be found here

To install CAESAR, be sure you have all requirements met as found here and run

$ git clone https://github.com/dnarayanan/caesar.git
$ cd caesar
$ python setup.py install
